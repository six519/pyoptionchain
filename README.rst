pyoptionchain
=============

Google Option Chain + Tk + Pandas DataFrame 

Working Demo
============

http://youtu.be/CjesHhBJwAo

Manual Installation
===================
::

	pip install -r requirements.txt
	./setup.py install

Installing Using PyPI
=====================
::

	pip install pyoptionchain

Running Test
============
::

	./tests.py

Running The App
===============
::

	pyoptionchain
